RoRAG-Trust: A Citation-Verifiable RAG Framework for Romanian1. OverviewRoRAG-Trust este un framework de Retrieval-Augmented Generation (RAG) dezvoltat pentru a aborda problema halucinațiilor în sistemele de Question Answering pentru limba română. Spre deosebire de pipeline-urile RAG standard, RoRAG-Trust introduce un Citation Verification Module care validează fiecare afirmație generată împotriva surselor recuperate.  
2. Main ContributionsNovel Algorithm (ATA-RAG): Un algoritm de tip Adaptive Trust-Aware RAG care fuzionează scorurile de relevanță, entalment și consistență pentru a estima încrederea.  Abstention Policy: Un mecanism care permite sistemului să se abțină de la a răspunde atunci când probele sunt insuficiente sau contradictorii, reducând rata de halucinație.  Benchmarking: Un set de date original pentru evaluarea acurateței factuale în limba română.  
3. ArchitectureProiectul este structurat modular pentru a asigura reproductibilitatea și extensibilitatea:  retrieval/: Implementarea unui HybridRetriever (BM25 + FAISS) pentru recuperare semantică și lexicală.  verification/: Nucleul cercetării, incluzând ClaimSplitter și CitationVerifier (NLI-based).  evaluation/: Module pentru calculul Faithfulness și analiza eficienței.  experiments/: Scripturi dedicate pentru Ablation Studies și Robustness Testing.  
4. Methodology & ExperimentsProiectul include un studiu de ablație (Ablation Study) pentru a măsura impactul fiecărei componente asupra performanței. Comparăm:  Baseline: RAG standard (fără verificare).RoRAG-Trust: Pipeline-ul complet (cu verificare NLI și abstinență).

6. Ethics & PrivacySistemul este conceput cu un focus pe transparență, oferind sursa pentru fiecare afirmație generată. Nu sunt stocate date cu caracter personal în timpul procesului de inferență.
